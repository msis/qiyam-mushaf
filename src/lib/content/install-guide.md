## Install Qiyam Mushaf

Installing the app gives you a native-app experience, offline access and a home screen icon.

### Safari (iOS / iPadOS / macOS)

1. Tap the **Share** button (□↑).
2. Scroll down and tap **"Add to Home Screen"**.

### Chrome / Edge (Android)

1. Tap the browser menu (⋮).
2. Select **"Add to Home screen"** or **"Install app"**.

### Chrome / Edge (Desktop)

Click the **install icon** in the address bar, or open the browser menu (⋮) and choose **"Install Qiyam Mushaf"**.

### Other Browsers

Not all browsers support PWA installation. If you don't see an install option, try using **Chrome**, **Safari**, **Edge** or **Firefox**.
